Sintaxe JavaScript

Literais

Valores fixos para representar dados primários, como números, strings, booleanos, arrays, objetos e expressões regulares.

Valores fixos para representar dados primários, como números, strings, booleanos, arrays, objetos e expressões regulares. Mais detalhes podem ser encontrados na documentação.

Variáveis

Usadas para armazenar dados; declaradas com var, let ou const, podendo ser modificadas.

São espaços de armazenamento de dados que podem ser modificados durante a execução do programa. São declaradas com as palavras-chave var, let ou const e podem conter diferentes tipos de dados.

Operadores

Realizam operações em operandos, incluindo aritméticos, de comparação e lógicos.

São símbolos ou palavras reservadas que realizam operações em operandos. Existem operadores aritméticos (como +, -, *, /), de comparação (como ==, !=, <, >) e lógicos (como &&, ||).

Expressões

Combinações de valores, variáveis e operadores avaliáveis para produzir um valor.

São combinações de valores, variáveis e operadores que são avaliadas para produzir um valor. Podem ser simples (como 2 + 2) ou complexas (como (a * b) / (c - d)).

Palavras-chave

Definem a estrutura e lógica do código, como if, else, for, function, entre outras.

São palavras reservadas que têm significados especiais na linguagem JavaScript. Elas definem a estrutura e a lógica do código, como if, else, for, function, entre outras.

Comentários

Trechos ignorados pelo interpretador; úteis para documentar e anotar o código.

São trechos de texto que são ignorados pelo interpretador JavaScript. São úteis para documentar o código, fazer anotações e explicar partes específicas do programa.

Tipos de dados

Números, strings, booleanos, arrays, objetos, funções, null e undefined.

São as diferentes categorias de valores que podem ser manipulados em JavaScript. Incluem números, strings, booleanos, arrays, objetos, funções, null e undefined.

Funções

Blocos de código reutilizáveis que executam tarefas específicas.

São blocos de código reutilizáveis que executam tarefas específicas. São definidas com a palavra-chave function e podem receber parâmetros e retornar valores.

Identificadores

Nomes para variáveis, funções, etc., começam com letra, sublinhado ou cifrão.

São nomes dados a variáveis, funções, objetos, etc. Eles devem começar com uma letra, sublinhado (_) ou cifrão ($), seguido de letras, números, sublinhados ou cifrões. São utilizados para identificar e acessar diferentes elementos no código JavaScript.
